# Program
- Program:
```cpp
#include <iostream>

int main(void)
{
	int myNumber = 16;
	int myNumberSize = sizeof(myNumber);
	
	std::cout << myNumberSize << std::endl;
	
	return 0;
}
```

![[sizeof-compile.png]]
- The **sizeof** function returns the size in bytes
